### E2E Test

Untuk Menjalankan antarmuka pengguna grafis (GUI) Cypress.

```sh
npx cypress open 
```
untuk menjalankan pengujian Cypress dalam mode non-interaktif atau mode headless.

```sh
npx cypress run
```
